COMPLETED PROJECTS

CODING & DESIGN

SCHOOL RANKING APP

Tools: HTML, CSS, SASS, Bootstrap, JavaScript, AOS, FontAwesome, PHP, SQL, Adobe Photoshop

The assumption of this project was to create a fully responsive website leading to the ranking of schools and universities. The site has, among others, the following functionalities: login and registration panel, detailed and general search, pagination, the ability to add comments and ratings, tracking selected schools, separate subpages for users and administrators. Passwords in the database are hashed by PHP functions and protected by Google reCAPTCHA tool and PHP htmlentities functions.

MEMORY GAME

Tools: HTML, CSS, SASS, JavaScript, FontAwesome, Adobe Photoshop

The Memory Game project is an electronic implementation of the world-famous game. During the creation of this project, I expanded my knowledge of JavaScript, including work and processing of arrays, and DOM manipulation methods. The game allows you to choose one of three prepared sets of images, while the duration of one game is a maximum of 35 seconds. The application is designed for stationary and mobile devices.
